API Reference¶

Aerosol Distributions¶

Aerosol Distribution Class

opcsim.AerosolDistribution([label]) Define an aerosol distribution.

AerosolDistribution Methods

opcsim.AerosolDistribution.add_mode(n, gm, gsd) Add a mode to the distribution.
opcsim.AerosolDistribution.pdf(dp[, base, ...]) Evaluate and return the probability distribution function at particle diameter dp.
opcsim.AerosolDistribution.cdf([dmin, dmax, ...]) Evaluate and return the cumulative probability distribution function between dmin and dmax.

Visualization¶

viz.distplot(distribution[, x, weight, ...]) with_modes only works for an individual weight

Utility Functions¶

opcsim.load_distribution(label) Load sample distributions as described by S+P Table 8.3.
