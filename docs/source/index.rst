============================================

opcsim is a Python module that allows you to easily create simulated optical particle
 counters (OPCs) along with simulated or real aerosol distributions to determine
 how they behave and respond to perturbations.

For a brief introduction and tutorial, please visit the introduction.

To see the code or report a bug, please visit the GitHub repository.
